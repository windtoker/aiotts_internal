Audio Samples from "AIOtts : All-in-One pitch-adjustable end-to-end TTS without external aligner"

Abstract:

"Issues existing in text-to-speech field have been resolved with several ideas including end-to-end architecture, pitch adjustable implementation and external alignment estiamtor for non-autoregressive models. However, there exists no solution tackling all issues in one solution at once. We propose pitch-adjustable end-to-end TTS without external aligner that generates naturally sounding speeches with simpler training steps by learning all at once. Duration prediction, pitch estimation and wave generation tasks are done at once to eliminate the inefficiency in the training and the inference steps. Compared to the state-of-the-art VITS in the inference stage, normalizing flow modules have been removed which accelerates the speed of audio generation. While VITS focuses on generating audios stochastically by sampling random latent variable in latent space, in AIOtts we can input the pitch transform factor to adjust the pitch level of each token we input as text. Thus we can manually cooridinate the prosody of the generated audios by adjusting the pitch factor in the inference stage."
